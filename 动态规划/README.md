## 计算各个位数不同的数字的个数
如果已经知道n位的结果，如何获得n+1位的结果呢，可以这样考虑。
对于n+1位的数字有两种可能:
> 前边的n位已经有了重复，那么最后一位0-9都可以  
> 前边的n位数字是没有重复的，那么最后一位可以是这n位数字中的任何一位
